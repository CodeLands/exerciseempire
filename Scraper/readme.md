## Setup:
<ul>
  <li>npm install</li>
  <li>npm install typescript</li>
  <li>npm install axios cheerio</li>
  <li>npm install pg</li>
  <li>npx install</li>
</ul>

## Tips:
<ul>

  <li>npx tsc -init</li>
</ul>

## Run:
<ul>
  <li>npx tsc</li>
  <li>node src/scraper.js</li>
</ul>
